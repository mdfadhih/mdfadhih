## 🏆 Featured Projects

### 🤖 [DocuChat — AI-Powered Document Q&A](https://github.com/mdfadhih/Docuchat) &nbsp;[![Live Demo](https://img.shields.io/badge/Live%20Demo-▶-e8ff47?style=flat-square&labelColor=0e0e0f)](https://docuchat-navy.vercel.app)
**Tech:** React · TypeScript · Node.js · Gemini AI · Supabase pgvector · Vercel · Render

A full-stack **RAG (Retrieval-Augmented Generation)** application. Upload any PDF and chat with it using natural language — answers are grounded in your document, not hallucinated.

- 🧠 Built complete RAG pipeline: PDF parsing → chunking → vector embeddings → cosine similarity search → streaming LLM response
- ⚡ Streaming responses token by token using Server-Sent Events
- 🎯 Dynamic question suggestions generated from document content using Gemini AI
- 🗄️ Vector storage with Supabase pgvector (768-dim Gemini embeddings)
- 🚀 Deployed with CI/CD — auto-deploys on every git push via Render + Vercel
